## Setup

- Clone [repository ini](https://github.com/muhangga/indonesian-insights-decoded)

```console
git clone https://github.com/muhangga/indonesian-insights-decoded.git

```

- Buat database Postgres terlebih dahulu, buat dengan nama database :

```console
decoded-insight-test

```

- Masuk ke directory dan jalankan project,

```console
- example : cd 1-sum
- go run main.go

```
